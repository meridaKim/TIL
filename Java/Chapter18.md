# Chapter 18
## 데이터 입출력
>참고 문헌 : 이것이 자바다

### 18.1 입출력 스트림
* 자바는 입력 스트림(데이터 들어옴)과 출력 스트림(데이터 나감)으로 데이터를 입출력한다.
* `스트림`은 **단방향**으로 데이터가 흐르는 것을 말한다.
* 바이트 스트림(InputStream, OutputStream) : 그림, 멀티미디어, 문자 등 모든 종류의 데이터를 입출력할 때 사용
* 문자 스트림(Reader, writer) : 문자만 입출력할 때 사용

### 18.2 바이트 출력 스트림
